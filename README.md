🌍 Atlas Explorer
მარტივი საიტი მსოფლიოს ქვეყნების შესახებ. შეგიძლია მოძებნო ქვეყანა სახელით ან დედაქალაქით, გაფილტრო რეგიონის მიხედვით, დაალაგო მოსახლეობით ან ფართობით და ნახო თითოეულის დეტალები — დროშა, ენა, ვალუტა, დროის სარტყელი. მოწონებული ქვეყნები გულის ღილაკით ინახება რჩეულებში, პირდაპირ ბრაუზერში (localStorage), სერვერზე არაფერი იგზავნება.
გვერდები

index.html — მთავარი: ძიება, რეგიონის ფილტრი, სორტირება, ქვეყნების ბადე
country.html — ერთი ქვეყნის დეტალები (დროშა, ენა, ვალუტა, დროის სარტყელი, Google Maps-ის ბმული)
favorites.html — შენახული ქვეყნები
about.html — პროექტის აღწერა და გამოყენებული წყაროები

ფუნქციონალი

ცოცხალი ძიება — შედეგები ფილტრდება ბეჭდვის დროსვე
რეგიონის dropdown (Africa, Americas, Asia, Europe, Oceania)
სორტირება სახელით, მოსახლეობით ან ფართობით, ორივე მიმართულებით
რჩეულების სისტემა — localStorage-ზე დაფუძნებული, სერვერის გარეშე
სრულად რესპონსივი, 320px-დან 1024px-მდე, ბურგერ მენიუთი მობილურზე
Cookie-ს შეტყობინება, ჰედერის ფონი სქროლზე, scroll-to-top ღილაკი

ტექნოლოგიები
სუფთა HTML5/CSS3 (custom properties, Flexbox/Grid — ბიბლიოთეკის გარეშე) და ვანილა JavaScript (fetch, async/await, jQuery და Bootstrap JS არსად გამოიყენება). სექციების ანიმაციისთვის — ScrollReveal ბიბლიოთეკა. ფონტები — Poppins და Inter, Google Fonts-იდან.
მონაცემები
ქვეყნების ინფორმაცია მოდის countries.dev API-დან, დროშები — flagcdn.com-იდან. ორივე თავისუფალია და გასაღებს არ საჭიროებს.
შეზღუდვა
რჩეულები ინახება მხოლოდ იმ ბრაუზერში, სადაც დაამატე — მოწყობილობებს შორის არ სინქრონდება.
