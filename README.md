# Atlas Explorer 🌍

მარტივი საიტი მსოფლიოს ქვეყნების შესახებ. შეგიძლია მოძებნო ქვეყანა სახელით ან დედაქალაქით, გაფილტრო რეგიონის მიხედვით, დაალაგო მოსახლეობით ან ფართობით და ნახო თითოეულის დეტალები — დროშა, ენა, ვალუტა, დროის სარტყელი და ა.შ. რომელიც მოგეწონება, შეგიძლია გულის ღილაკით შეინახო ფავორიტებში, ეს ბრაუზერშივე ინახება (localStorage), სერვერზე არაფერი იგზავნება.

გვერდები: მთავარი (`index.html`), ქვეყნის დეტალები (`country.html`), ფავორიტები (`favorites.html`) და About (`about.html`).

მონაცემები მოდის [countries.dev](https://countries.dev/) API-დან, დროშები — [flagcdn.com](https://flagcdn.com/)-იდან. სუფთა HTML/CSS/JS-ზეა დაწერილი, ბიბლიოთეკებისა და framework-ების გარეშე (გარდა ScrollReveal-ისა, სექციების ანიმაციისთვის).


Live: `https://njeladze.github.io/atlas-explorer-/`
